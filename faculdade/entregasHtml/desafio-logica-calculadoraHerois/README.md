# Desafio: Calculadora de Heróis

Descrição
- Projeto de lógica para uma calculadora de heróis. Interface em HTML/CSS e lógica em JavaScript para combinar/comparar atributos, calcular resultados simples de combate e demonstrar princípios de Programação Orientada a Objetos.

Tecnologias
- HTML, CSS, JavaScript

Como executar
- Abrir `index.html` no navegador.
- Ou rodar um servidor local (opcional):
    - Python 3: `python -m http.server 8000`
    - Extensão VSCode: Live Server

Estrutura do projeto
- `index.html` — interface principal
- `styles/` — arquivos CSS (se aplicável)
- `js/` — lógica da calculadora (se aplicável)
- `assets/` — imagens e ícones (se aplicável)
- `README.md` — este arquivo

Funcionalidades típicas
- Entrada de atributos dos heróis (força, defesa, habilidade etc.)
- Cálculos de dano/resultado simples
- Visualização comparativa entre dois heróis
