# Clínica Médica

Projeto de estudo em Programação Orientada a Objetos para a disciplina de faculdade.

## Descrição

Aplicação web simples para uma clínica médica, com funcionalidades básicas de cadastro e visualização de pacientes, médicos e agendamentos.

## Tecnologias

- HTML
- CSS
- JavaScript (se houver)
- Possível uso de JavaScript no front-end

## Estrutura do projeto

- `index.html` — página principal
- `css/` — estilos
- `js/` — scripts
- `assets/` — imagens e recursos

## Como usar

1. Abra o `index.html` no navegador.
2. Navegue pelas páginas para cadastrar e visualizar dados.
3. Edite os arquivos conforme necessário para implementar funcionalidades.

## Objetivo

Desenvolver habilidades em:
- construção de interfaces web
- organização de arquivos
- conceitos básicos de POO aplicados ao front-end
