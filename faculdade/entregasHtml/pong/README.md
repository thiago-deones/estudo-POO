
# Pong

Um clássico jogo de Pong implementado em HTML, CSS e JavaScript.

## 🎮 Sobre o Jogo

Pong é um dos primeiros videogames da história. Este projeto é uma recriação moderna do jogo original, onde dois jogadores competem para mover as raquetes e impedir que a bola passe por suas laterais.

## 🚀 Como Jogar

- **Jogador 1 (W/S)**: Controle a raquete esquerda para cima e para baixo
- **Jogador 2 (↑/↓)**: Controle a raquete direita para cima e para baixo
- Bata a bola de volta para o adversário
- Primeiro a atingir o número de pontos necessário vence

## 📋 Requisitos

- Um navegador web moderno (Chrome, Firefox, Edge, Safari)

## 🛠️ Instalação

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` no seu navegador
3. Comece a jogar!
