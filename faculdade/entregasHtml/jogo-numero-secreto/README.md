# Jogo do Número Secreto

## Descrição
Jogo interativo onde o objetivo é adivinhar um número secreto gerado aleatoriamente.

## Como Jogar
1. O computador escolhe um número secreto entre 1 e 100
2. Você tenta adivinhar o número
3. Receba dicas: "maior" ou "menor"
4. Continue até acertar o número

## Tecnologias
- HTML
- CSS
- JavaScript

## Como Executar
Abra o arquivo `index.html` no seu navegador.
