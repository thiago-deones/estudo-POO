# 🎮 Cadastro e Ataques de Heróis

Este é um projeto simples em JavaScript que simula a criação de personagens de jogos (heróis) e seus respectivos métodos de ataque. O código utiliza conceitos fundamentais de Orientação a Objetos (OO), como classes, construtores e estruturas de decisão (`switch/case`).

## 🚀 Funcionalidades

- **Criação de Heróis:** Permite instanciar personagens definindo `nome`, `idade` e `tipo`.
- **Ataque Dinâmico:** O método `.atacar()` identifica automaticamente o tipo do herói e exibe uma mensagem personalizada no console:
  - **Mago:** Ataca usando magia.
  - **Guerreiro:** Ataca usando espada.
  - **Monge:** Ataca usando artes marciais.
  - **Ninja:** Ataca usando shuriken.

## 🛠️ Tecnologias Utilizadas

- **JavaScript (ES6+)**
- **Node.js** (para execução local via terminal)

## 📋 Como Executar o Projeto

Certifique-se de ter o [Node.js](https://nodejs.org) instalado no seu computador.

1. Abra o seu terminal (`cmd` ou terminal do VS Code).
2. Navegue até a pasta onde o arquivo está salvo.
3. Execute o comando abaixo (substitua `nome-do-arquivo.js` pelo nome real do seu arquivo):

```bash
node nome-do-arquivo.js
```

### 💻 Exemplo de Saída no Console

Ao rodar o script atual, a saída esperada será:

```text
O mago, atacou usando usou magia!
O ninja, atacou usando usou shuriken!
```
